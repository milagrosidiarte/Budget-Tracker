# Rastreador de Presupuesto Personal (Next.js + Supabase)

Proyecto final de Programación IV.

## Stack principal

- **Frontend / Backend**: Next.js 14 (App Router)
- **Base de datos y Auth**: Supabase (PostgreSQL + Auth)
- **Deploy**:
  - Frontend/Fullstack: Vercel
  - DB + Auth: Supabase

## Objetivo

Implementar un rastreador de presupuesto personal que cumpla:

- Requisitos generales de la app web con autenticación y autorización.
- Requisitos avanzados de Programación IV.
- Requisitos específicos del caso de estudio "Rastreador de Presupuesto Personal / Finanzas (Budget Tracker)".
